# Twitch Clone

Twitch Clone is 6-screen app built using flutter. 
It has following features:
- Email & Password Authentication
- Go Live with thumbnail and title
- Switch Camera at live streaming
- Mute/Unmute mic at live streaming
- Screensharing at live streaming (runs only on web)
- Real-time chat at live streaming
- Real-time record of live users
- Ending livestream
- Joining livestream from live users
